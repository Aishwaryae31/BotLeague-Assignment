# 🤖 BotLeague – Robotics Arena Platform

A modern, responsive frontend implementation of the **BotLeague Robotics Competition Platform**, developed as part of the BotMakers Internship Assignment.

## 📖 Overview

BotLeague is a futuristic robotics competition ecosystem where participants can:

* Build teams
* Compete in robotics events
* Earn national rankings
* Showcase achievements
* Connect with the robotics community

This project recreates the provided UI design using modern frontend technologies with a focus on responsiveness, clean architecture, and user experience.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aishwaryae31/BotLeague-Assignment.git
```

Navigate to the project:

```bash
cd BotLeague-Assignment
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
